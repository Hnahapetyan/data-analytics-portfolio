# Driving Safety KPI Dashboard

## Business Task
Build an interactive dashboard to analyze driving safety metrics and fatalities by state,
with a focus on KPIs, targets, and trends.

## Tools Used
- Power BI
- Excel

## Key Features
- Single-select state slicer
- KPI cards and gauge with target and thresholds
- Conditional formatting based on performance levels
- Filled map synchronized with KPI colors
- KPI visual with trend and target

## Result
The dashboard enables quick assessment of driving safety performance
and supports data-driven decision-making.
