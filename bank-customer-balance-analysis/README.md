# Bank Customer Balance Analysis Dashboard

## Business Task
Create a banking dashboard to analyze customer balances
by education level, job classification, housing status, and marital status.

## Tools Used
- Power BI
- Excel

## Key Features
- KPI cards for total, male, and female balances
- Balance breakdown by education, job classification, and marital status
- Interactive education slicer
- Clear separation between KPI and distribution visuals

## Result
The report supports customer segmentation analysis
and financial portfolio insights.
